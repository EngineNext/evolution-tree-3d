# 進化の樹形図 ― Arbor Vitae

生命の進化系統樹を 3D でたどるインタラクティブな博物図鑑サイトです。共通祖先（LUCA）から哺乳類・鳥類までの系譜を立体的に表示し、動物ノードをクリックすると図鑑説明がポップアップします。白背景・古紙質感のアンティーク調デザイン。

## 特徴

- **3D 系統樹**: 共通祖先から枝分かれした全 36 ノードの立体ツリー（`three` / `@react-three/fiber` / `@react-three/drei`）
- **クリックで図鑑**: 金の輪をまとった動物ノードをクリックすると、和名・学名・出現年代・解説・豆知識を表示
- **操作**: ドラッグで回転、ホイールで拡大縮小、ゆっくり自動回転
- **デザイン**: 白背景・古紙テクスチャ・明朝セリフ・金の装飾罫のアンティーク調

## 動かし方

Node.js（LTS 版）が必要です。

```bash
npm install      # 初回のみ。必要な部品をインストール
npm run dev      # 開発サーバーを起動
```

表示された `http://localhost:5173/` をブラウザで開いてください。

## その他のコマンド

```bash
npm run build    # 本番用にビルド（dist/ が生成される）
npm run preview  # ビルド結果をローカルで確認
npm run lint     # ESLint でコードをチェック
```

## 技術構成

- React 19 + Vite
- Three.js（react-three-fiber / drei）による 3D 描画
- カスタム CSS によるアンティーク調スタイリング
